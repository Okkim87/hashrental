# HashRental

Rent a real Bitaxe Gamma Bitcoin miner in minutes.

HashRental is a hardware rental service that allows customers to rent a hosted Bitaxe Gamma ASIC miner and connect it to their own Bitcoin mining pool.

## Features

- Dedicated Bitaxe Gamma (1.3 TH/s)
- Remote activation
- Electricity included
- Hosted hardware
- Your own mining pool
- Mining rewards paid directly to your own wallet
- Mobile friendly
- Responsive design
- GitHub Pages compatible

## Rental Plans

### Starter

- €9
- 7 Days

### Extended

- €19
- 14 Days

## How It Works

1. Choose a rental plan.
2. Complete payment using SumUp.
3. Send your mining pool information.
4. Your Bitaxe is activated remotely.
5. Mining rewards are paid directly by your mining pool to your Bitcoin wallet.

## Replace the SumUp Links

Open:

```
index.html
```

Replace

```
YOUR_7_DAY_SUMUP_LINK
```

with your 7-day payment link.

Replace

```
YOUR_14_DAY_SUMUP_LINK
```

with your 14-day payment link.
